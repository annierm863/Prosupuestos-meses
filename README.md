# Presupuesto Personal - Versión Ultimate

Aplicación web para gestión de presupuesto personal con Firebase.

## Estructura del Proyecto

```
/
├── index.html      # Página principal
├── app.js          # Lógica de la aplicación
├── styles.css      # Estilos
└── README.md       # Este archivo
```

## Configuración de GitHub Pages

Asegúrate de que:
1. Los archivos estén en la raíz del repositorio (o en la carpeta `docs/`)
2. GitHub Pages esté configurado para servir desde la raíz o desde `docs/`
3. Las rutas en `index.html` sean relativas: `./app.js` y `./styles.css`

## Solución de Problemas

Si ves errores 404:
1. Verifica que todos los archivos estén en el repositorio
2. Revisa la consola del navegador (F12) para ver qué archivo está fallando
3. Asegúrate de que las rutas en el HTML sean correctas
